## Configuração

```bash
git clone https://github.com/peimelo/heroes-crud.git
cd heroes-crud

# installation of dependencies
npm install

# run the project
ng serve
```

Abra o navegador no endereço `http://localhost:4200`.

## Deploy

Visite esse site em produção no endereço `https://heroes-crud.vercel.app`.
